About Course

Front-end web devlopment is everything involved in programming the user interface of a web application .Typically it refers to the Hypertext Markup Language(HTML),Cascading Style Sheets(CSS) and JavaScript portion of web site production as opposed to the database or server-side programming.It encompaases everything from building a simple html page of HTML text to creating complex ,responsive HTML5 websites designed to be accessed via various different browsers,devices and screeen sizes.
